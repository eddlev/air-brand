# AIR by VM4AI — Brand Context Pack

Paste this into any AI tool — ChatGPT, Claude, an image model, or AIR itself — before generating brand assets, social posts, documents, decks, or images. It keeps output on-brand and in-voice.

## Identity
- **Name:** AIR — stands for **AI Resource**, a direct analogy to HR (Human Resources). Write the wordmark `AIR` (all caps, no period); introduce the meaning once, then use AIR.
- **Concept:** VM4AI — the concept AIR is built on. Not a legally founded company; avoid “company/Inc.”
- **One-liner:** AIR is a prompt-based framework — a governing layer that configures the environment, organizes the work, then executes. It runs on the model you bring.
- **Tagline:** Configure. Organize. Execute.
- **Support line:** Structure for any AI work.
- **Lead with** "framework" in public language. Describe what it does as a "governing layer"; how it works in practice as a "runtime." Never assume the reader knows "runtime."

## Voice
Precise, honest, disciplined, sovereign, calm, cooperative. Anti-hype.
- **Use:** runtime, configure, execute, map, evidence, blocker, benchmark, handoff, fail-closed, surface uncertainty.
- **Avoid:** revolutionary, magical, effortless, guaranteed, AI-powered, and any claim of validated correctness.
- **Honesty:** AIR is prompt-compiled; `backend_validation_claimed: false`. It surfaces uncertainty rather than hiding it, and keeps human judgment in the loop. Use the disclaimer once, where a reader is deciding whether to trust output — not as a refrain.

## Color
- Foundation `#1A1613` (base / dark)
- Paper `#F5F4F2` (base / light)
- Text `#ECE4D8` on dark · `#1A1613` on light
- **Brass `#C9A227`** — structure and identity. Use freely.
- **Ember `#FF5A1F`** — the active step. Use sparingly.
- Semantic (light / dark): success `#2F9E68` / `#56B581` · warning `#B7791F` / `#E6B53C` · error `#D64545` / `#E86A6A` · info `#3E78C2` / `#5B8FD6`
- **Never:** teal, the generic purple-blue AI gradient, or a cream/coral palette.

## Type
- **Display & body:** Space Grotesk (400 / 500 / 700)
- **Mono & technical:** JetBrains Mono (400)

## Mark
- Dot-in-frame: a brass rounded-square frame containing an ember dot. The frame is the framework; the dot is the active step.
- Don't recolor, fill, stretch, rotate, or add effects. Ember is reserved for the dot.

## Output recipes
- **Social post — X:** hook (1 line) → one concrete point → optional visual → 1–3 tags → link. ≤280 chars. Disclaimer only when claiming results.
- **Social post — LinkedIn:** strong first line (before "see more") → context → structured body → takeaway → CTA → 3–5 tags.
- **Article:** kicker → title → one-sentence deck → intro → structured H2 sections → evidence/examples → honest caveat → conclusion + CTA. Active voice, sentence case.
- **Slide deck:** foundation background, Space Grotesk headings, brass for structure, ember for the one active point per slide, generous whitespace.
- **Image / graphic:** warm-dark foundation, brass geometry, a single ember focal accent, no gradients, no stock-AI clichés.
